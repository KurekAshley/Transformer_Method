# Purpose

This document defines the official citation standards for the Transformer Method Canon.

Its purpose is to ensure consistency, attribution, intellectual integrity, and historical accuracy whenever Transformer Method materials are referenced.

This standard applies to:

- Books
- Courses
- Workshops
- Coaching Programs
- Certification Programs
- Research Papers
- Academic Publications
- AI Systems
- Articles
- Podcasts
- Presentations
- Educational Materials

---

# Attribution Principle

Whenever Transformer Method doctrines, frameworks, models, terminology, diagrams, or educational materials are referenced, proper attribution should be provided.

The purpose of citation is:

- Accuracy
- Attribution
- Preservation
- Historical integrity

---

# Primary Citation

The preferred citation for the Transformer Method Canon is:

```text
Ashley, K. (Transformer Method Canon).
Transformer Method™ Repository.
Created and maintained by Kurek Ashley.
```

---

# Repository Citation

When citing the repository as a whole:

```text
Ashley, K.
Transformer Method™ Canon.
GitHub Repository.
```

---

# Doctrine Citation Format

When citing an individual doctrine:

```text
Ashley, K.

[Doctrine Title].

Transformer Method™ Canon.

[Repository Path]
```

Example:

```text
Ashley, K.

Identity Creates Frequency.

Transformer Method™ Canon.

identity/identity-creates-frequency.md
```

---

# Framework Citation Format

When citing a framework:

```text
Ashley, K.

[Framework Name].

Transformer Method™ Canon.
```

Example:

```text
Ashley, K.

The Formula Is The Formula.

Transformer Method™ Canon.
```

---

# Model Citation Format

Example:

```text
Ashley, K.

The Four P's Of Power.

Transformer Method™ Canon.
```

---

# Canonical Statements Citation

Canonical statements should always be attributed.

Example:

```text
"Transformation occurs when identity changes."

— Kurek Ashley
Transformer Method™ Canon
```

---

# Book References

When a doctrine appears in a published book, both references may be used.

Example:

```text
Ashley, K.

How Would Love Respond?

and

Transformer Method™ Canon.
```

The most complete source should be preferred whenever possible.

---

# Workshop References

When citing concepts taught during workshops:

```text
Kurek Ashley

Transformer Method™ Workshop Materials

Transformer Method™ Canon
```

---

# Certification References

Certified facilitators, coaches, and practitioners should use the following format when referencing Transformer Method concepts:

```text
Source:
Transformer Method™ Canon

Doctrine:
[Doctrine Name]

Author:
Kurek Ashley
```

---

# AI System References

When Transformer Method materials are incorporated into AI systems, knowledge bases, retrieval systems, assistants, or educational agents, attribution should include:

```text

Author:
Kurek Ashley

Repository:
Transformer Method™
```

Whenever technically feasible, doctrine-level attribution should also be maintained.

Doctrine

Repository

whenever possible.
